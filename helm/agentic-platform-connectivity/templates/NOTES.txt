Prerequisite check (run BEFORE installing the agentic platform):
  kubectl get crd gateways.gateway.networking.k8s.io
  kubectl get crd agentgatewayparameters.agentgateway.dev
  kubectl get crd mcpservers.muster.giantswarm.io

This chart installs NO CRDs. The CRDs its CRs consume are app-owned: each
component (agentgateway, kagent, muster, agent-sandbox) ships its own CRDs in its
chart's crds/ dir. Install the component charts that own the CRDs this wiring
references first (or let the agentic-platform meta-package order it for you — it
renders this chart as a HelmRelease that dependsOn the agentgateway + kagent
component releases), then install this release.

  # agentgateway.dev CRDs ride the agentgateway component chart; kagent.dev CRDs
  # ride the kagent component chart; muster.giantswarm.io CRDs ride muster.
  helm install {{ .Release.Name }} \
    oci://gsoci.azurecr.io/charts/giantswarm/agentic-platform \
    --version <agentic-platform-version> -n {{ .Release.Namespace }}

Verify the data-plane Deployment renders after the controller picks up the Gateway:
  kubectl -n {{ .Release.Namespace }} get gateway,deployment -l gateway.networking.k8s.io/gateway-name={{ .Values.gateway.name }}
