Prerequisite check (run BEFORE installing the agentic platform):
  kubectl get crd gateways.gateway.networking.k8s.io
  kubectl get crd agentgatewayparameters.agentgateway.dev
  kubectl get crd mcpservers.muster.giantswarm.io

This chart does NOT install any CRDs. Required CRDs are owned by sibling
releases that must exist first. Encode ordering via Flux HelmRelease
`spec.dependsOn`, or follow the raw-helm sequence:

  helm install agentgateway-crds \
    oci://cr.agentgateway.dev/charts/agentgateway-crds --version v1.2.1 \
    -n {{ .Release.Namespace }} --create-namespace
  helm install muster-crds \
    oci://gsoci.azurecr.io/charts/giantswarm/muster-crds \
    --version <muster-crds-version> -n {{ .Release.Namespace }}
  helm install {{ .Release.Name }} \
    oci://gsoci.azurecr.io/charts/giantswarm/agentic-platform \
    --version <agentic-platform-version> -n {{ .Release.Namespace }}

Verify the data-plane Deployment renders after the controller picks up the Gateway:
  kubectl -n {{ .Release.Namespace }} get gateway,deployment -l gateway.networking.k8s.io/gateway-name={{ .Values.gateway.name }}
